# GIT

- It is a version control system which helps us to track, make changes for our applications.

# GitHub
- Its cloud based platform which keeps all our codes and files in the form of folders called Repositories (repo).

# Commands

` git clone <--some repo link--> `

  - This will clone the remote repository to your local machine.

` git status `

  - This will check the status of your code i.e. tracked, commits, changed or unchanged.
  - there r 4 things that comes under status.
    <br>
   1. untracked : new files that r not in git yet.
   2. modified  : files that r changed but not added.
   3. staged    : files r ready to be committed.

   4. unmodified: means unchanged.
